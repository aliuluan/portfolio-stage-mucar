# Guide d'installation du Démonstrateur Mu-CAR
## Prérequis : Node.js (v18+) et PostgreSQL (v15+)
1. Cloner le dépôt : git clone [...]
2. Installer les dépendances frontend et backend : npm install
3. Initialiser la base de données via le script : psql -U user -f mucar_gen.sql
4. Lancer le serveur de développement : npm run dev
